UlamMethod.ulam_binner — Method
ulam_binner(traj, domain)

Selects and executes the appropriate binning algorithm for the given trajectories and domain. Intersects the resulting polygons with the domain so that the returned polygons are clipped to it. Returns Vector{UlamPolygon{Float64}}.
